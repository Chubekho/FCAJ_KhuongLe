Throughout my 12-week internship in the **BOOTCAMP FIRST CLOUD AI JOURNEY (FCAJ)** program, from **April 20, 2026** to **July 10, 2026**, I had the opportunity to engage systematically with the Amazon Web Services ecosystem — starting from the most foundational concepts (IAM, VPC, EC2) all the way to more advanced services such as Serverless, Containers, CI/CD, and the security/operational principles required at a production scale.

I directly built and deployed the workshop project **AI Content Generator Platform** — a multi-tenant SaaS platform that leverages Generative AI to automate marketing content production, deployed on real AWS infrastructure (a dedicated VPC, RDS Multi-AZ, EC2 Auto Scaling, asynchronous processing via SQS/Lambda, a CDN, and WAF), along with a fully automated CI/CD pipeline built with GitHub Actions. Through this process, I significantly improved my skills in cloud system architecture design, full-stack development (Node.js/Express, React), a least-privilege security mindset, and diagnosing and resolving real-world issues that only surface once a system goes to production — something a purely theoretical learning environment could rarely provide.

In terms of work ethic, I consistently tried to investigate problems on my own before seeking help, took the deadlines of each phase seriously, and continuously reflected on my own shortcomings to improve week over week.
